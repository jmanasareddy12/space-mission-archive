# 🚀 Space Mission Archive System

A web application built with **Flask** and **MySQL** to manage space missions, spacecraft, and astronaut data efficiently.  

---

## 🛠️ Tech Stack
- **Backend:** Flask  
- **Database:** MySQL  
- **Frontend:** HTML, CSS, JavaScript  
- **Other Tools:** Flask-Login, Flask-WTF
